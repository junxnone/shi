# 莜面

  - 和面
  - 搓莜面
  - 蒸莜面
  - 莜面羊肉汤
  - 莜面凉汤

## 和面

  - 莜面 一半
  - 开水一半，烧开状态倒入搅拌，冷却
  - 根据需要的硬度调节水量

## 搓莜面

### 条状莜面

  - 取一小块莜面，团成圆锥形
  - 手掌握成手刀状，使用手刀刀刃部分搓圆锥尖头边缘，并向前拉伸
  - 面板长一点能够更方便搓出均匀并长的莜面
