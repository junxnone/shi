# 白灼芥兰/菜心

  - 芥兰/菜心
  - 生抽
  - 蒜沫
  - 藤椒油(可选)

## 步骤

  - 起锅烧水，煮沸
  - 芥兰/菜心 放入水中煮熟后摆盘
      - 可以先放茎部，再放叶子
  - 锅中倒入生抽, 藤椒油少许，少许水，烧开
  - 倒入蒜沫煮沸
  - 将料汁浇入盘中
