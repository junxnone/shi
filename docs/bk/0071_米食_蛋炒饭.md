# 蛋炒饭

## 分离式蛋炒饭

  - 蛋液加盐搅匀
  - 煎蛋尽量碎一点
  - 停火，放入米饭{尽量干一点}
  - 炒透米饭🍚
  - 加入小葱碎，炒出香味，出锅

## 包裹式蛋炒饭

  - 鸡蛋只取蛋黄
  - 蛋黄加盐搅碎后倒入米饭
  - 搅拌，让米粒外面包裹一层蛋液
  - 起锅烧油，炒米饭
  - 炒透加入小葱，炒香出锅
