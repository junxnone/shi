## 茶的分类
- 绿茶
- 黄茶
- 白茶
- 青茶
- 红茶
- 黑茶


### 按照 `汤色/味道` 排序

```mermaid

graph LR
    subgraph 淡
    A((绿茶))
    B(黄茶)
    C(白茶)
    A --> B --> C
    end
    subgraph 适中
    D(青茶)
    end
    subgraph 浓
    E(红茶)
    F((黑茶))
    E --> F
    end
    C --> D
    D --> E
```
